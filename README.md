# 🚨 Fake News AI Generator

A modern AI-powered web application that generates humorous and satirical fake news headlines.  
The app also displays trending real-world news topics using web scraping and presents them with a clean dark-themed UI.

Built using **Python and Streamlit**, this project demonstrates concepts of **Generative AI, web scraping, and interactive UI development**.

---

#  Features

### 🎲 Random Fake News Generator
Generates random satirical news headlines using predefined templates.

### 🤖 AI Headline Generator
Users can enter a topic and the system generates a humorous AI-style fake news headline.

### 🌍 Trending News Section
Fetches trending news headlines using **Google News RSS** and displays them as modern cards with images.

### 🌙Modern Dark UI
Clean and visually appealing interface with cards, images, and responsive layout.
